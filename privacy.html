Privacy Policy for Screenshot2Pin

Effective Date: June 15, 2026

Screenshot2Pin provides workflow automation tools to facilitate streamlined visual research and asset management via the Pinterest platform.

1. Data Collection and Retention

During the current application phase, all operations run strictly end-to-end within the user's localized environment. The application does not collect, harvest, or store personal data, screen captures, or images on external servers. Temporary session tokens utilized for authentication are securely processed locally. If a user disconnects their account, all temporary local authentication parameters are instantly cleared.

2. Data Usage and Third-Party Sharing

The application interacts exclusively with official Pinterest Developer Platform endpoints to manage board synchronization and authorized Pin creation. We do not sell, rent, lease, or redistribute user content or Pinterest-derived data to any third-party marketing networks or external entities.

3. Platform Disclaimer

This application utilizes official Pinterest API frameworks to enable automation features. This product and its developer brand operate independently and are not endorsed by, sponsored by, or affiliated with Pinterest Inc.
